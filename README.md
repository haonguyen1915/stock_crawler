# stock_crawler



#### Install

```shell
pip install scrapy
```

#### Init project

```shell
scrapy startproject stocks
```

#### Run

```shell
cd stocks
# test 1
scrapy crawl stock1 -o data1.json

# test 2
scrapy crawl stock2 -o data2.json

# test 1
scrapy crawl stock3 -o data3.json

# test 1
scrapy crawl stock4 -o data4.json
```
